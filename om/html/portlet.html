    Portlet Colors

    Portlet color is controlled by specifying a color class, for example to make a red portlet you would write:

    {{ h.portlet('Blue Portlet', cls='blue') }} {{ h.portlet('Dark Blue Portlet', cls='dark-blue') }} {{ h.portlet('Red Portlet', cls='red') }} {{ h.portlet('Gray Portlet', cls='gray') }} {{ h.portlet('Light Gray Portlet', cls='light-gray') }} {{ h.portlet('Yellow Portlet', cls='yellow') }} {{ h.portlet('White Portlet', cls='white') }}

    Portlet Corners

    Portlets come with all 4 corners rounded by default. If you want to specify that only some corners be rounded, you need to add corner classes to the portlet's class.

    You can also mix and match the corner classes to get funky combinations

    {{ h.tr(['corner-top', 'Top two corners are are rounded, bottom are not']) }} {{ h.tr(['corner-bottom', 'Bottom two corners are are rounded, top are not']) }} {{ h.tr(['corner-left', 'Left two corners are are rounded, right are not']) }} {{ h.tr(['corner-right', 'Right two corners are are rounded, left are not']) }} {{ h.tr(['corner-tl', 'Only top left corner is rounded']) }} {{ h.tr(['corner-bl', 'Only bottom left corner is rounded']) }} {{ h.tr(['corner-tr', 'Only top right corner is rounded']) }} {{ h.tr(['corner-br', 'Only bottom right corner is rounded']) }} {{ h.tr(['corner-none', 'No rounded corners']) }}
